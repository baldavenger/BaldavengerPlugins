# BaldavengerPlugins
OFX Plugins for DaVinci Resolve (macOS, Win10, Linux)

Place (plugin).ofx.bundle in Plugins directory:

(for macOS) 
/Library/OFX/Plugins

(for win10) 
C:\Program Files\Common Files\OFX\Plugins

(for linux) 
/usr/OFX/Plugins
